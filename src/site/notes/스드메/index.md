## ✅ 결정 전 체크리스트

- [ ] 각 스튜디오 인스타 최신 화보 톤 비교 (따뜻 vs 화사 vs 클래식)
- [ ] 촬영 의상 벌 수 / 원본·수정본 제공 조건 확인
- [ ] 케이트린 피팅비(55,000원) · 헬퍼비(25만원) 별도 확인
- [ ] 셀럽비비 원본+수정본 필수 44만원 등 **추가비용** 견적 포함 여부 확인
- [ ] 히엘 담당 직급(원장/부원장/실장)별 가격 차이 확인
- [ ] 야외씬·토탈 헬퍼비 등 옵션 추가금 확인
- [ ] 이동 동선(스튜디오↔드레스↔메이크업) 및 본식 지역 헬퍼비 확인
