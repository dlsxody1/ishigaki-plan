## ✈️ 1. 공항 도착 & ATM

| 일본어 | 발음 | 한국어 |
| --- | --- | --- |
| すみません、ATMはどこですか？ | 스미마셍, ATM와 도코데스카? | 실례합니다, ATM이 어디예요? |
| 到着ロビーはこちらですか？ | 토차쿠 로비와 코치라데스카? | 도착 로비가 이쪽인가요? |
| 五万円を引き出したいです | 고만엔오 히키다시타이데스 | 5만엔을 인출하고 싶습니다 |
| このカードは使えますか？ | 코노 카도와 츠카에마스카? | 이 카드 사용할 수 있나요? |
| 国際カードは使えますか？ | 코쿠사이 카도와 츠카에마스카? | 해외 카드 되나요? |
| 手数料はいくらですか？ | 테스료와 이쿠라데스카? | 수수료는 얼마예요? |

> [!warning] 예외 케이스
> **ATM 오류 시**
> - 「このカードは使えません」= 이 카드는 사용할 수 없습니다
> - 「取引ができません」(토리히키가 데키마셍) = 거래할 수 없습니다
> → 류큐은행 ATM으로 이동하거나, 화면에서 「English」 버튼 눌러서 영어 전환
> 
> **잔액 부족 시**
> - 「残高不足です」(잔다카 부소쿠데스) = 잔액이 부족합니다
> → 트래블월렛 앱에서 충전 후 재시도

## 🚐 2. 렌터카 셔틀 & 렌터카

| 일본어 | 발음 | 한국어 |
| --- | --- | --- |
| レンタカーのシャトルバスですか？ | 렌타카노 샤토루바스데스카? | 렌터카 셔틀버스인가요? |
| ○○レンタカーを予約した○○です | ○○렌타카오 요야쿠시타 ○○데스 | ○○렌터카 예약한 ○○입니다 |
| 予約番号は○○です | 요야쿠 반고와 ○○데스 | 예약번호는 ○○입니다 |
| 国際免許証です | 코쿠사이 멘쿄쇼데스 | 국제면허증입니다 |
| 満タン返しですか？ | 만탄가에시데스카? | 가득 채워서 반납인가요? |
| ガソリンスタンドはどこですか？ | 가소린 스탄도와 도코데스카? | 주유소가 어디예요? |
| レギュラー満タンでお願いします | 레규라 만탄데 오네가이시마스 | 보통 휘발유 가득 넣어주세요 |
| この車のガソリンの種類は何ですか？ | 코노 쿠루마노 가소린노 슈루이와 난데스카? | 이 차의 기름 종류가 뭐예요? |
| 返却場所はここですか？ | 헨캬쿠 바쇼와 코코데스카? | 반납 장소가 여기인가요? |

> [!warning] 예외 케이스
> **차에 문제가 생겼을 때**
> - 「パンクしました」(판쿠 시마시타) = 펑크났어요
> - 「エンジンがかかりません」(엔진가 카카리마셍) = 시동이 안 걸려요
> - 「ぶつけてしまいました」(부츠케테 시마이마시타) = 부딪혀버렸어요
> - 「事故を起こしました」(지코오 오코시마시타) = 사고가 났어요
> → 렌터카 업체 긴급연락처 + 110 (경찰) / 119 (소방·구급)
> 
> **네비게이션**
> - 「ナビを使ってもいいですか？」(나비오 츠캇테모 이이데스카?) = 네비 써도 되나요?
> - 「電話番号で検索できますか？」(덴와방고데 켄사쿠 데키마스카?) = 전화번호로 검색 가능한가요?
> → 일본 네비는 전화번호 입력으로 목적지 설정 가능!

> [!warning] 예외 케이스
> **얼리 체크인 / 레이트 체크아웃**
> - 「早めにチェックインできますか？」(하야메니 체크인 데키마스카?) = 일찍 체크인 가능한가요?
> - 「レイトチェックアウトは可能ですか？」(레이토 체크아웃와 카노데스카?) = 늦은 체크아웃 가능한가요?
> - 「追加料金はかかりますか？」(츠이카 료킨와 카카리마스카?) = 추가 요금이 드나요?
> 
> **에어컨/시설 문제**
> - 「エアコンが効きません」(에아콘가 키키마셍) = 에어컨이 안 돼요
> - 「お湯が出ません」(오유가 데마셍) = 온수가 안 나와요
> - 「鍵が開きません」(카기가 아키마셍) = 열쇠가 안 열려요

| 別々にお願いします | 베츠베츠니 오네가이시마스 | 따로따로 계산해주세요 |
| 持ち帰りできますか？ | 모치카에리 데키마스카? | 포장 가능한가요? |

> [!warning] 예외 케이스
> **만석일 때**
> - 「満席です」(만세키데스) = 만석입니다
> - 「何時頃空きますか？」(난지고로 아키마스카?) = 몇 시쯤 자리 나나요?
> - 「近くに似たお店はありますか？」(치카쿠니 니타 오미세와 아리마스카?) = 근처에 비슷한 가게 있나요?
> 
> **알레르기/못 먹는 것**
> - 「アレルギーがあります」(아레루기가 아리마스) = 알레르기가 있어요
> - 「○○は入っていますか？」(○○와 하잇테이마스카?) = ○○ 들어있나요?
> - 「○○を抜いてください」(○○오 누이테 쿠다사이) = ○○ 빼주세요
> - 에비 (えび) = 새우 / 카니 (かに) = 게 / 타마고 (卵) = 계란 / 코무기 (小麦) = 밀
> 
> **영업시간 관련**
> - 「ラストオーダーは何時ですか？」(라스토 오다와 난지데스카?) = 라스트오더 몇 시예요?
> - 「まだ注文できますか？」(마다 추몬 데키마스카?) = 아직 주문 가능한가요?
> - 「準備中」(준비추) = 준비 중 (아직 안 열었음)

> - 「船に戻りたいです」(후네니 모도리타이데스) = 배로 돌아가고 싶어요
> 
> **날씨/취소**
> - 「今日のツアーは催行されますか？」(쿄노 츠아와 사이코 사레마스카?) = 오늘 투어 진행되나요?
